## To run this project

-  cd to into server and 
``
Run npm install
``

-  cd to into client and 
``
Run npm install
``

- Get your API KEY at https://beta.openai.com/account/api-keys

- Create .env file inside server folder and paste your generated key

#inside server
``
Run npm run server
``
#inside client
``
Run npm run dev
``
